document.write(), innerHTML, innerText, value, nodevalue

document.write

document.write是直接写入到页面的内容流，如果在写之前没有调用document.open, 浏览器会自动调用open。每次写完关闭之后重新调用该函数，会导致页面被重写。即第一次调用document.write会将当前页面的内容清空, 其后再次调用会在原来的内容上继续添加, 如果之后调用了document.close()或将页面关闭, 再次使用document.write会再次将页面内容清空并输出, 识别标签

innerHTML

innerHTML是dom元素的一个属性，代表该元素的html内容。你可以精确到某一个具体的元素来进行更改。如果想修改document的内容，则需要修改document.documentElement.innerElement, 识别标签

innerText

innerText设置的内容不识别标签, 获取内容是从起始位置到终止位置的内容, 但它去除Html标签, innerText并非W3C标准属性, 原为ie特有属性,经测试到目前为止已经得到主流浏览器最新版本的支持

value

用来设置和获取具有value属性标签的value值, 主要用于表单中

nodevalue

对于文本节点，nodeValue 属性包含文本。
对于属性节点，nodeValue 属性包含属性值。
nodeValue 属性对于文档节点和元素节点是不可用的。
nodeValue是W3C标准属性
