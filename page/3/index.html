By Heart Walker

發表於 2016-02-16

清除浮动

在css布局时经常会用到浮动,但浮动脱标后会带来许多问题,下边列出几种清除浮动的方法

1.：给父盒子一个高度
2.：clear:left| right| both

　　不推荐 基本没人用吧

3.：给父盒子设置overflow:hidden

　　有定位的话不推荐 应为可能超出盒子 看需求使用

4.：使用伪类
.clearfix:after{
    content:"";
    height:0;
    line-height:0;
    visibility:hidden;
    display:block;
    clear:both;
}
.clearfix{
    *zoom:1;
}

　　目前通用方法

5. : 双伪元素法
.clearfix:before, .clearfix:after {
    content: "";
    display: table;
}

.clearfix:after {
    clear: both;
}

.clearfix {
    *zoom: 1; /*IE/7/6*/
}

　　应该是目前最好的方法了吧
